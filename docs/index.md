# Software Design/Architecture & Data Engineering

Bienvenue dans ce cours dédié au **Software Design**, à l'**Architecture Logicielle** et au **Data Engineering**.

Ce cours a pour objectif de vous fournir les bases solides et les meilleures pratiques pour concevoir des systèmes robustes, évolutifs et performants.

## Software Design

- [Type Hint](cours-1-type-hint.md) : Comprendre et utiliser le typage statique en Python pour améliorer la qualité et la robustesse du code.
- [Data Structure](cours-2-datastructure.md) : Maîtriser les structures de données en Python (types numériques, strings, booléens, enums, collections) avec des exemples pratiques et des considérations de performance.
- [Class et Dataclass](cours-3-class-et-dataclass.md) : Comprendre les classes, les méthodes dunder, les dataclasses et leurs fonctionnalités avancées (order, frozen, slots, default_factory), ainsi que l'alternative Pydantic.
